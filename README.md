# 🏠 Bengaluru House Price Prediction

A machine learning-based web application that predicts house prices in Bengaluru based on various features.

## 📋 Features

- Predicts house prices in lakhs (₹ Lakhs)
- Takes into account:
  - Area Type (Super built-up, Built-up, Plot, Carpet)
  - Location
  - BHK (Number of bedrooms)
  - Total Square Feet
  - Number of Bathrooms
  - Number of Balconies
  - Availability Status
  - Society Name (optional)

## 📊 Model Performance

- **Algorithm:** Random Forest Regressor
- **R² Score:** ~85-90%
- **Features:** 8
- **Data Source:** Bengaluru Housing Dataset

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **ML Framework:** Scikit-learn
- **Data Processing:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
- **Deployment:** GitHub + Streamlit Cloud
